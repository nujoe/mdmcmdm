# layout_analysis.md - 브루탈리즘 레이아웃 및 스타일링 분석

사용자께서 업로드해주신 레이아웃 스케치 이미지와 `imageslide.md` 문서를 기반으로, 전체 레이아웃 구조와 브루탈리즘(Brutalistic) 디자인 컨셉을 분석 및 설계합니다.

---

## 1. 이미지 분석 및 UI 구성 요소

제시된 이미지 스케치는 크게 다음과 같은 화면 구성과 정렬 기준을 가집니다.

1. **상단 네비게이션/유틸리티 컨트롤 (좌측 상단)**
   * **버튼 1: `작업 아카이브`**
     * **[중요 예외 사항]** 버튼 스타일링 border에 **rounded corner**를 적용합니다.
   * **버튼 2: `정보`**
     * 마찬가지로 border에 **rounded corner**를 적용합니다.
   * *이 외의 모든 요소는 직각 코너(no rounded corners)를 가져야 합니다.*

2. **중앙 이미지 슬라이더 영역 (`div`) (가운데 넓은 영역)**
   * `imageslide.md`에서 언급된 **Pure CSS 3D animated carousel**이 위치합니다.
   * 이 영역과 안의 이미지들, 프레임은 **둥근 모서리(rounded corner)가 적용되지 않아야 합니다.**
   * **1px 두께의 검정색 테두리(border: 1px solid black;)**를 사용해 거칠고 정직한 브루탈리즘 스타일로 마감합니다.

3. **하단 애플 독(Dock) 영역 (`div`) (하단 중앙)**
   * `goals.md`에서 구현하기로 한 **Apple Dock Navigation Bar**가 위치합니다.
   * 아이콘 확대(Magnification) 파도 효과는 그대로 유지되지만, 독의 전체적인 컨테이너와 아이콘 테두리 등은 브루탈리즘 스타일에 맞추어 **직각(no rounded corners)** 및 **1px 검정 테두리**로 감싸집니다.

---

## 2. 브루탈리즘(Brutalist) 스타일 가이드라인

* **둥근 모서리 배제:** `border-radius: 0;`을 기본값으로 사용합니다. (단, 이미지 좌측 상단의 `작업 아카이브` 및 `정보` 버튼은 예외적으로 rounded corner 적용)
* **단호한 테두리:** 주요 컨테이너와 이미지 슬라이더, 개별 이미지 카드 프레임에 `border: 1px solid #000000;`을 명시합니다.
* **평면적 입체감 (High Contrast):** 부드러운 그라데이션 대신, 원색(혹은 아주 강한 대비의 흑백/그레이톤)과 1px 검정 보더를 이용하고, 그림자 효과는 부드러운 blur 대신 `box-shadow: 4px 4px 0px #000;` 형태의 Neo-Brutalist 하드 섀도우를 고려합니다.
* **폰트:** 시스템 산세리프 폰트나 디폴트 모노스페이스(monospace) 폰트를 활용해 장식 없는 투박함을 유지합니다.
