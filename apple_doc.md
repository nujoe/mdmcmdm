# Apple Dock Navigation Bar - 기능 및 인터랙션 분석

이 문서는 CodePen에 공유된 Osmo의 'Apple Dock Navigation Bar' 코드 중 프론트엔드 시각적 효과와 사용자 인터랙션(기능적 부분)을 분석하고 정리한 문서입니다.

---

## 1. 주요 시각적 특징 (Visual Features)

* **macOS Dock 스타일 디자인:** 애플 맥(Mac) 운영체제 하단에 위치한 Dock(독) 인터페이스를 웹으로 완벽하게 구현한 네비게이션 바입니다.
* **앱 아이콘 툴팁 (Tooltip):** 아이콘(Notion, Asana, Slack, Figma 등) 위에 마우스를 올리면 숨겨져 있던 해당 앱의 이름이 툴팁 형태로 나타납니다. (`.nav-item__tooltip`)
* **커스텀 커서 (Custom Cursors):** * 기본 화면(`body`)과 클릭 가능한 요소(`a`, `button`)에 시스템 기본 마우스 포인터 대신, 외부 SVG 이미지를 활용한 커스텀 커서를 적용하여 독특한 사용자 경험(UX)을 제공합니다.

---

## 2. 핵심 인터랙션: 확대(Magnification) 파도 효과

이 코드의 가장 핵심적인 기능은 특정 아이콘에 마우스를 올렸을 때, **해당 아이콘과 주변 아이콘들이 물결치듯 부드럽게 커지는 효과**입니다. 이는 JavaScript의 인덱스 계산과 CSS의 조합으로 이루어집니다.

### 동작 알고리즘 (JavaScript & CSS 시너지)

1.  **이벤트 리스너 등록:** * 모든 아이콘 요소(`.nav-item`)를 선택한 뒤, 마우스가 요소 안으로 들어올 때(`mouseenter`) 작동하는 이벤트를 부여합니다.
2.  **타겟 요소 (직접 호버된 요소) 확대:** * 마우스가 직접 닿은 아이템에는 `hover` 클래스를 추가하여 가장 큰 비율로 확대(Scale up) 시킵니다.
3.  **인접 요소(Siblings) 거리별 동적 클래스 부여:** * `toggleSiblingClass`라는 커스텀 도우미 함수를 사용하여 현재 호버된 아이콘의 인덱스(Index)를 기준으로 좌우 요소를 추적합니다.
    * **1단계 인접 요소 (거리 ±1):** 호버된 아이템 바로 양옆의 아이콘에는 `sibling-close` 클래스를 부여합니다. (중간 비율로 확대)
    * **2단계 인접 요소 (거리 ±2):** 바로 양옆의 그 다음 아이콘들에는 `sibling-far` 클래스를 부여합니다. (작은 비율로 확대)
4.  **시각적 결과:** * CSS에 정의된 클래스별 크기 차이와 부드러운 전환(Transition) 효과가 맞물려, 마우스가 이동할 때마다 돋보기로 보는 듯한 자연스러운 곡선 형태의 파도(Wave) 애니메이션이 완성됩니다.

---

## 3. 코드 구조 요약

* **HTML:** `<nav>` 태그 내부 `<ul>`과 `<li>` 리스트를 사용하여 아이콘(이미지)과 툴팁(텍스트)을 그룹화하여 구조적으로 배치했습니다.
* **CSS:** 전체적인 색상과 커스텀 커서를 지정하며, 인터랙션 시 부여되는 클래스(`.hover`, `.sibling-close`, `.sibling-far`)에 따른 크기 변화(`transform: scale` 등)와 애니메이션 속도를 제어합니다.
* **JavaScript:** 마우스 호버 이벤트를 감지하고, 현재 아이템을 중심으로 -2 부터 +2 위치에 있는 형제 요소들을 수학적으로 계산하여 CSS 클래스를 추가/제거(Toggle)하는 뇌쇄적인 역할을 담당합니다.
